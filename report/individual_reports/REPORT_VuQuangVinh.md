## II. Debugging Case Study (10 Points)

*Analyze a specific failure event you encountered during the lab using the logging system.*

- **Problem Description**: Agent caught in an infinite loop. It successfully generated the audio file in Iteration 1, but instead of stopping, it continued to rename and regenerate files until hitting the `max_iterations=3` limit, resulting in a `Timeout / Max Steps Exceeded` error (occurred 7 times in 17 test runs).
- **Log Source**: Telemetry Data (`logs/*.jsonl`) - Aggregate Reliability: 29.4%.
- **Diagnosis**: Context Drift (Attention Drift). As the ReAct conversation history grew, the `gemini-2.5-flash` model lost focus on the `Observation: Thành công` feedback and reverted to trying to solve the original user prompt again.
- **Solution**: Updated the `system_prompt` with a Strict Constraint (Guardrail): *"⚠️ QUY TẮC QUYẾT ĐỊNH: Nếu trong phần 'Observation:' trước đó có chữ 'Thành công. Kết quả file lưu tại...', nhiệm vụ ĐÃ HOÀN THÀNH. Bạn KHÔNG ĐƯỢC gọi thêm công cụ nào nữa. Bạn BẮT BUỘC phải kết luận bằng định dạng: Final Answer:..."*.

## III. Personal Insights: Chatbot vs ReAct (10 Points)

*Reflect on the reasoning capability difference.*

1.  **Reasoning**: The `Thought` block acted as a crucial semantic translator. It allowed the model to break down a vague natural language request ("make a chill beat") into a strict JSON payload (`{"tempo": 80, "waveform": "lofi", "key": "C"}`) required by the backend audio compiler.
2.  **Reliability**: The Agent performed significantly *worse* than the Chatbot on pure theory questions (e.g., "What notes are in a C major chord?"). It hallucinated Python code tools to print the notes, causing high latency (~3-5 seconds) and wasting tokens, whereas the Chatbot answered instantly.
3.  **Observation**: Environmental feedback is a double-edged sword. While it allows the agent to self-correct if a tool fails (e.g., invalid JSON), without strict stopping rules, the LLM treats successful observations as just another context piece rather than a termination signal.

*How would you scale this for a production-level AI agent system?*
